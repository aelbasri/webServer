Introduction

The Hypertext Transfer Protocol (HTTP) is the foundation of data communication for the World Wide Web. At 1337 Coding School, the Webserv project challenges students to build their own HTTP server from scratch using C++. This project is a deep dive into networking, socket programming, and web server architecture.

About 1337 Coding School

1337 is a unique coding school that focuses on peer-to-peer learning and project-based education. Located in Morocco, it is part of the 42 Network, which includes campuses worldwide. The curriculum emphasizes practical skills, creativity, and problem-solving.

Webserv Project Details

The goal of the Webserv project is to create a fully functional HTTP server that can handle requests from web browsers. The server must support features like:

  • Handling HTTP methods: GET, POST, and DELETE.
  • Serving static files and handling dynamic content via CGI.
  • Managing multiple client connections using poll(), select(), or epoll().
  • Parsing a configuration file to customize server behavior.

Project Requirements

The project has strict requirements to ensure robustness and performance:

  • The server must be non-blocking and handle multiple clients efficiently.
  • It must support file uploads and error pages.
  • The server should be compatible with modern web browsers.
  • Stress testing is required to ensure the server remains available under heavy load.
