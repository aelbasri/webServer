Quick Start Guide

1

Configure

Set up your server configuration file

server {
  listen 8080;
  root /var/www/html;
  index index.html;
  
  location / {
    methods GET POST;
    autoindex on;
  }
}
2

Compile

Build the server using make

$ make
$ ./webserv42 config.conf
3

Test

Verify your server is working

$ curl -X GET http://localhost:8080
$ curl -X POST -d "data=test" http://localhost:8080
$ curl -X DELETE http://localhost:8080/file.txt
