Project Overview

Webserv is an advanced C++ project that challenges you to build your own HTTP server from scratch. This project helps you understand the inner workings of web servers and the HTTP protocol that powers the internet.

You'll implement a fully functional web server that can serve static content, handle various HTTP methods, process form submissions, execute CGI scripts, and much more - all while ensuring high performance and reliability.

Core Features

  • Custom configuration file parsing
  • Non-blocking I/O operations
  • HTTP request parsing and validation
  • Proper HTTP response generation
  • Multiple virtual servers on different ports
  • File serving with proper MIME types
  • Directory listing functionality
  • Error page customization

HTTP Methods

  • GET - Retrieve resources
  • POST - Submit data and upload files
  • DELETE - Remove resources
  • Custom route configuration
  • HTTP redirections

Technical Requirements

  • Written in C++ 98
  • Non-blocking I/O with poll() or equivalent
  • Single process architecture
  • Resilient error handling
  • Compatible with web browsers
  • CGI script execution
  • File upload handling
